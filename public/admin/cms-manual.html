一、登录CMS后台

1.1 访问地址

在浏览器地址栏输入以下地址进入CMS后台：

后台地址：https://您的域名/admin/

1.2 登录方式

本系统使用 Netlify Identity 进行身份认证：

  1. 点击页面中的 "Login with Netlify Identity" 按钮
  2. 输入您的注册邮箱和密码
  3. 首次使用需要由管理员邀请并设置密码
提示：如果您忘记密码，请点击登录页面的"忘记密码"链接，系统会发送重置邮件到您的注册邮箱。

1.3 邀请新用户

  1. 登录 Netlify 管理面板
  2. 进入 Site Settings → Identity
  3. 点击 "Invite users" 输入新用户邮箱
  4. 新用户会收到邀请邮件，点击链接设置密码即可

三、媒体资产库管理

重要原则：所有模块中使用的图片和视频应先上传到媒体资产库，再在其他模块中引用。这样可以：

  • 避免重复上传相同文件
  • 统一管理SEO Alt Text，引用时自动继承
  • 通过标签快速筛选和查找资源

3.1 添加新媒体资产

  1. 进入 "媒体资产库" 模块
  2. 点击 "New 媒体资产"
  3. 填写以下字段：
字段说明必填
资产名称资源的描述性名称，如"工厂全景航拍"是
SlugURL友好标识符，如"factory-aerial-view"是
资产类型选择：图片/视频/文档是
文件上传上传本地文件二选一
外部URL填入YouTube/Vimeo等外部链接二选一
SEO Alt Text图片的SEO替代文本是
标签分类标签，可多选是

3.2 标签分类说明

  • 产品演示 - 产品使用和展示图片/视频
  • 工厂环境 - 工厂、生产线、仓库照片
  • Hero Banner - 首页和各页面顶部横幅图
  • 证书资质 - CE、ISO等证书扫描件
  • 客户案例 - 客户现场施工照片
  • 技术图解 - 产品爆炸图、技术示意图
  • 博客配图 - 博客文章内配图
  • 团队照片 - 团队和公司活动照片

4.1 创建新产品

  1. 进入 "产品管理" 模块
  2. 点击 "New 产品"
  3. 按顺序填写各部分字段

4.2 基本信息

产品名称产品的完整名称Cordless Concrete Nail Gun

产品分类选择所属分类混凝土钉枪
简短描述产品简要介绍（1-2句）20V锂电无绳混凝土钉枪
详细描述支持Markdown格式的详细介绍—
主图产品主图（建议800x800px）—

4.3 硬件参数（展开填写）

点击 "硬件参数" 展开面板，填写以下工业级参数：

4.4 媒体增强

  • 技术爆炸图：上传产品的零件爆炸图，用于前端交互展示
  • 演示视频：添加15-30秒产品演示短视频（支持多个）
  • 产品图集：添加多张产品展示图片（建议含Alt Text）

4.5 关联推荐与PDF规格书

  • 关联推荐产品：输入关联产品的Slug，实现"看了又看"推荐
  • PDF规格书：上传产品规格PDF，供前端用户直接下载
提示：产品的SEO设置在最下方折叠面板中。如果留空，系统会自动使用产品名称和描述。建议为重点产品手动填写SEO字段以获得更好的搜索引擎排名。

五、博客与成功案例

5.1 创建博客文章

  1. 进入 "博客管理" 模块
  2. 点击 "New 博客文章"
  3. 选择分类（产品指南/行业洞察/商业/公司新闻/技术/成功案例）
  4. 使用Markdown编辑器撰写正文

5.2 发布成功案例

成功案例是博客的一种特殊分类。创建时：

  1. 分类选择 "成功案例 / Case Studies"
  2. 展开下方 "案例研究字段" 面板
  3. 填写行业分类、客户痛点、解决方案等专属字段
  4. 上传现场图集和视频
案例专属字段说明
行业分类建筑工程/基础设施/制造业/农业/能源
客户痛点描述客户遇到的主要问题
解决方案详情详细描述NailTask提供的解决方案
现场图集施工现场照片（支持多张）
案例视频YouTube/Vimeo链接
写作建议：每篇文章建议800-1500字，配3-5张图片。使用二级标题(##)和三级标题(###)组织内容结构，有助于SEO。

六、信任背书库（证书管理）

管理企业获得的各类资质认证证书。

6.1 添加证书

  1. 进入 "信任背书库" 模块
  2. 点击 "New 证书"
  3. 选择证书分类（CE/ISO/RoHS/UL/TUV/SGS/其他）
  4. 填写证书编号、颁发机构、有效期
  5. 上传证书扫描件和PDF
注意：请及时更新证书到期日期，过期证书应及时替换或标注。建议设置日历提醒，在证书到期前30天更新。

七、资源下载中心

管理可供客户下载的PDF文件，如产品手册、参数表、产品目录等。这些资源可用作营销获客的"Lead Magnet"。

7.1 添加新资源

  1. 进入 "资源下载中心" 模块
  2. 点击 "New 资源"
  3. 选择文件分类
  4. 上传PDF文件
  5. 可选：上传自定义封面图（建议 400x560px）
  6. 填写文件大小和描述
  7. 如关联特定产品，填写产品Slug

7.2 文件分类说明

分类说明
产品手册完整的产品使用手册
参数规格表产品技术参数一览表
产品目录全线产品目录册
安装指南产品安装步骤说明

八、客户见证

8.1 添加客户评价

  1. 进入 "客户见证" 模块
  2. 点击 "New 客户见证"
  3. 填写客户基本信息（姓名、职位、国家为必填）
  4. 选择评价类型（文字/视频）
  5. 输入评价内容
  6. 上传客户头像（建议 200x200px 正方形）
提示："是否展示"开关可控制评价是否在前端显示。"排序权重"数字越大越靠前。

九、常见问题管理

9.1 添加FAQ

  1. 进入 "常见问题" 模块
  2. 点击 "New 问答"
  3. 填写问题和答案（答案支持Markdown格式）
  4. 选择分类（技术/物流/售后/订单/OEM/产品使用）
  5. 开启/关闭 "SEO结构化数据" 开关

9.2 SEO结构化数据说明

开启"SEO结构化数据"后，该FAQ将以 application/ld+json 格式输出到页面，这有助于：

  • 在Google搜索结果中显示Rich Snippet（丰富摘要）
  • 提升页面在搜索结果中的展示面积
  • 增加点击率（CTR）

十、全球服务网络

10.1 添加服务节点

  1. 进入 "全球服务网络" 模块
  2. 点击 "New 服务节点"
  3. 选择所在区域，填写国家、城市
  4. 填写联系人和联系方式
  5. 输入经纬度坐标（用于地图展示）
  6. 添加服务范围列表

10.2 获取经纬度坐标

方法：打开 Google Maps，右键点击目标位置，即可看到经纬度坐标。格式如：纬度 30.2741，经度 120.1551

十一、页面管理

页面管理模块用于编辑各固定页面的内容。

11.1 首页设置

  • Hero 标语：首页顶部大标题
  • Hero 副标题：标题下方说明文字
  • Hero 描述：详细描述段落

十二、站点设置

12.1 常规设置

在 "站点设置 → 常规设置" 中配置：

  • 站点名称 / Logo / Favicon
  • 分析追踪：GA4 ID、GTM ID、百度统计、Search Console验证码
  • 联系信息：邮箱、电话、WhatsApp、Skype、微信
  • CRM集成：选择CRM类型（小满Okki/企业邮箱/自定义API），配置API地址和通知邮箱
  • 多语言设置：选择默认语言和启用的语言列表

12.2 社交媒体

在 "站点设置 → 社交媒体" 中配置各社交平台链接。

12.3 CRM集成配置

系统支持三种CRM对接方式：

类型说明配置方式
小满 Okki CRM自动推送线索到小满CRM在Netlify环境变量中配置 CRM_PROVIDER=okki 和 CRM_API_ENDPOINT
企业邮箱新线索发送邮件通知配置 CRM_PROVIDER=email 和 CRM_NOTIFICATION_EMAIL
自定义API推送到自定义API接口配置 CRM_PROVIDER=custom 和 CRM_API_ENDPOINT
配置步骤：登录 Netlify → Site Settings → Environment Variables → 添加对应变量

十三、SEO配置指南

13.1 全站SEO

  • sitemap.xml：已自动生成，位于 /sitemap.xml
  • robots.txt：已自动配置，位于 /robots.txt
  • JSON-LD 结构化数据：已在全站嵌入Organization类型结构化数据

13.2 页面级SEO

每个产品、博客文章、页面都有独立的SEO设置面板：

字段说明建议
Meta Title搜索引擎标题50-60个字符，包含主要关键词
Meta Description搜索引擎描述120-160个字符，包含行动号召
Meta Keywords关键词（逗号分隔）5-10个相关关键词

13.3 图片SEO

所有图片的Alt Text会从媒体资产库自动继承。确保在上传到媒体资产库时填写有意义的Alt Text描述。

SEO最佳实践：
  • 每个页面使用唯一的Meta Title和Description
  • 在标题中自然包含目标关键词
  • 博客文章使用结构化标题（H2/H3）
  • 为所有图片提供描述性Alt Text
  • FAQ开启结构化数据以获取Rich Snippet

十四、内容发布工作流

14.1 内容编辑流程

  1. 登录CMS后台
  2. 选择要编辑的模块
  3. 创建新内容或编辑已有内容
  4. 使用"预览"功能检查效果
  5. 点击 "Publish" 按钮发布
  6. 等待约1-2分钟，网站自动更新

14.2 发布状态说明

状态说明
Draft（草稿）内容已保存但未发布，不会显示在前端
In Review（审核中）等待审核后发布
Ready（就绪）审核通过，可以发布
Published（已发布）内容已发布到网站前端

14.3 媒体库调用流程

标准流程：
  1. 先在"媒体资产库"上传图片/视频并填写Alt Text
  2. 在产品/博客等模块中，点击图片字段的"选择文件"
  3. 从已上传的文件中选择（路径会以 /uploads/ 开头）
  4. Alt Text会自动继承

十五、常见问题与故障排除

Q: 登录页面一直加载？

确认 Netlify Identity 已启用。进入 Netlify 管理面板 → Site Settings → Identity → 确认状态为 "Enabled"。

Q: 发布后网站没有更新？

发布后需要等待1-2分钟触发自动构建。可以在 Netlify 管理面板的 Deploys 页面查看构建状态。

Q: 图片上传失败？

  • 确认文件大小不超过 10MB
  • 支持格式：JPG、PNG、GIF、WebP、SVG
  • 建议使用英文文件名（避免中文和特殊字符）

Q: Markdown格式怎么用？

  • # 标题 - 一级标题
  • ## 标题 - 二级标题
  • **粗体** - 粗体文字
  • *斜体* - 斜体文字
  • [链接文字](URL) - 超链接
  • ![图片描述](图片URL) - 插入图片
  • - 列表项 - 无序列表
  • 1. 列表项 - 有序列表

Q: 如何备份内容？

所有内容都存储在Git仓库中，天然具备版本历史。如需回退，可在Git中查看历史版本。
